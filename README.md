# 🎯 Quiz App

A responsive and interactive Quiz App built using **HTML, CSS, and JavaScript**. The application allows users to answer multiple-choice questions, receive instant feedback, and view their final score. It is designed with a clean and user-friendly interface to provide an engaging quiz experience.

## 🚀 Features

- 📝 Multiple-choice quiz questions
- ✅ Instant answer validation
- 📊 Real-time score tracking
- 🔄 Restart quiz functionality
- 📱 Responsive design for desktop and mobile
- 🎨 Simple and modern user interface

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 💻 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Omkar130206/Quiz-Master-App.git
   ```

2. Open the project folder.

3. Run the project by opening `index.html` in your web browser.
